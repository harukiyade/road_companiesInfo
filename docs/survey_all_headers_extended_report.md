# fixed_csv_3 配下 CSV 徹底再調査レポート（拡張キーワード＋AI判断）

**調査対象**: `fixed_csv_3` フォルダ配下の全CSV（**除外**: `fixed_csv_3/later`）  
**対象ファイル数**: 182件  
**全CSVから抽出したユニークなカラム数**: 79

**調査手順**
1. 全182ファイルの全ヘッダーを抽出し、重複排除したユニークリストを作成
2. 部分一致キーワードで代表・役員系／株主・資本系／拠点・施設系を抽出
3. キーワードに該当しなくても「代表者・株主・拠点に関連しうる」カラムをAI判断で追加

---
