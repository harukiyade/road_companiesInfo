## インポート時の留意点

### 1. 列のずれ・欠損
- **業種列**: 「業種1〜4」「業種（細）」「業種-大/中/小/細」の複数パターンあり
- **住所関連**: 「会社郵便番号/会社住所」と「郵便番号/住所」の表記ゆれ
- **代表者関連**: 「代表者」と「代表者名」、「代表者郵便番号/住所」の有無

### 2. 重複列・空列
- グループ3: 郵便番号・住所が2回出現
- 複数グループで Unnamed 列や空列あり

### 3. エンコーディング
- **csv0115**: Shift-JIS（CP932）の可能性が高い。UTF-8で読むと文字化け

### 4. 値の整合性
- 設立: 「4月1日1980年」「1980-04-01」「20852（シリアル日付）」など形式が混在
- 直近決算年月: 「2022年9月1日」「45139」など
- 法人番号: 「9180000000000」のような仮値やプレースホルダの可能性

### 5. スキーマ変換の推奨マッピング
- 法人番号 ⇔ corporate_number
- 会社名 / 企業名 ⇔ name
- 都道府県 ⇔ prefecture
- 代表者名 / 代表者 ⇔ representative_name
- URL ⇔ url
- 業種1〜4 / 業種（細）/ 業種-大〜細 ⇔ 業種系カラム
- 郵便番号 / 会社郵便番号 ⇔ postal_code
- 住所 / 会社住所 ⇔ address
