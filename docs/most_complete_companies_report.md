# データ充実度ランキング：上位5社（検証用ターゲット企業）

**目的**: 企業DBの「肉付け（データ流し込み）」検証のため、情報の網羅性が高い企業を特定し、DB側の保持状況との比較に利用する。

**対象**: `fixed_csv_3` 配下の全CSV（`fixed_csv_3/later` を除く）  
**評価軸**: 次の5カテゴリの項目が可能な限りすべて埋まっている企業を優先  
1. 代表者・役員情報（代表者名、住所、誕生日、取締役リスト）

4. 業績・規模情報（売上、利益、決算年月、社員数）  
5. 事業内容・ネットワーク（業種、企業概要、取引先、取引先銀行、URL）

**調査方法**: Pythonスクリプト `scripts/find_most_complete_companies.py` で全ファイルを走査し、上記18項目の充実度でランキング。

## 補足

- **出典**: 5社とも `fixed_csv_3/unit_million/108.csv` の同一ファイルから取得した行が最高スコアとなりました。他ファイルに同一企業が存在する場合は、108.csv の行を「最も充実した1件」として採用しています。
- **スコア**: 18項目（代表者名・代表者住所・代表者誕生日・取締役／役員・株主・資本金・オフィス数・工場数・店舗数・直近売上・直近利益・直近決算年月・社員数・業種・概要／説明等・取引先・取引先銀行・URL）のいずれも非空の企業を 18/18 とし、そのうち上位5社を掲載しています。
- **検証時の利用**: 上記5社をDB上で企業名・住所または電話番号で検索し、代表者・株主・拠点・業績・事業内容の各項目が期待どおり取り込まれているかを確認してください。
